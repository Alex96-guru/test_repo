# Demo site

Многостраничный пример сайта для GitHub Pages и локального Node-сервера.

## Что есть в проекте

- `index.html` - главная страница
- `services.html` - страница услуг
- `projects.html` - страница проектов
- `contact.html` - страница контактов и формы
- `server.js` - локальный сервер и API
- `data/site.db` - SQLite-база, создается автоматически при запуске

## Локальный запуск

1. Убедись, что установлен Node.js 22+.
2. Выполни `npm start`.
3. Открой адрес, который покажет сервер, обычно `http://localhost:3000`.

## Функция с БД

При запуске через Node доступны API-маршруты:

- `GET /api/stats` - количество заявок и последние записи
- `POST /api/leads` - сохранение заявки в SQLite

Форма на `contact.html` использует эти маршруты.

## GitHub Pages

Статические страницы можно публиковать на GitHub Pages.

Важно: GitHub Pages не запускает `server.js`, поэтому форма там остается только демонстрацией интерфейса без записи в БД.

Чтобы включить Pages:

1. Запушь репозиторий на GitHub.
2. Открой `Settings -> Pages`.
3. В `Build and deployment` выбери `Deploy from a branch`.
4. Укажи ветку `main` и папку `/ (root)`.
5. Сохрани настройки.
